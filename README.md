# Actionhero Backend api login


## To install:
(assuming you have [node](http://nodejs.org/) and NPM installed)

`brew install redis`
`brew install mongodb`

Launch Redis on computer starts.

`$ ln -sfv /usr/local/opt/redis/*.plist ~/Library/LaunchAgents`

Start mongo db
--dbpath=/mongodb

`npm install`

## To Run:

`npm start`

## To Test:

`npm test`
